AWS Portfolio Deployment Architecture
=====================================

┌─────────────────────────────────────────────────────┐
│                    USERS                            │
│              (Global Locations)                     │
└────────────────┬────────────────────────────────────┘
                 │ HTTPS
                 ↓
┌─────────────────────────────────────────────────────┐
│          CloudFront Distribution                    │
│      (ddj6jvqxzj0e2.cloudfront.net)                │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │  Edge Locations (40+ worldwide)              │  │
│  │  - Caching (TTL optimized)                   │  │
│  │  - SSL/TLS termination                       │  │
│  │  - Compression (Gzip/Brotli)                 │  │
│  └──────────────────────────────────────────────┘  │
└────────────────┬────────────────────────────────────┘
                 │ OAC (Origin Access Control)
                 ↓
┌─────────────────────────────────────────────────────┐
│          S3 Bucket (Private)                        │
│      thato-portfolio-website                        │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │  Static Files:                               │  │
│  │  - index.html                                │  │
│  │  - style.css                                 │  │
│  │  - script.js                                 │  │
│  │  - images/                                   │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  Bucket Policy: CloudFront-only access              │
│  Public Access: BLOCKED ❌                          │
└─────────────────────────────────────────────────────┘

Security Flow:
- Internet → S3 Direct = 403 Forbidden ❌
- Internet → CloudFront → S3 = Success ✅
